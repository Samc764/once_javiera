# Como crear un repositorio en Github
-Necesitamos una cuenta de Github **Personal**
--
# Pasos
-Despues de iniciar sesión en tu cuenta de Github personal, le damos al icono `+` que esta en la esquina superior derecha y seleccionamos **New repository** 

-Ahora vamos a configurar los datelles del respositorio:

--Primero vamos a elegir un nombre para nuestro repositorio.

--Después agregamos una descripción **ESTE PASO ES OPCIONAL**.

--Elegimos si nuestro repositorio sera publico para que los demas lo vean o privado para que solo tú lo veas.

-Y ahora como ultimo paso pero no menos importante le damos en **Create repository**
